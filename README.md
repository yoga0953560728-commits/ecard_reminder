# E-Card Reminder System

This project is a Python-based reminder tool designed to help users manage their dormitory electricity card.

## Features
- Time-based reminder
- Pop-up notification (tkinter)
- Delay function (remind later)
- Daily reset system

## Description
The program continuously checks the current time and compares it with a preset reminder time. When the time matches, a notification window appears.
